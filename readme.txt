1.将该文件夹下的4个文件，一同拷贝到ubuntu中的某一个目录，如/mydata目录，ubuntu建立目录可采用如下命令
cd /
mkdir mydata

2.修改downpull.sh related.sh run_static三个文件的权限，可采用命令如下：
chmod 777 downpull.sh related.sh run_static

3.打开input.txt文档（通过右键找到gedit打开或者在命令行用vim打开），在里面对搜索条件进行配置：
第一行: 预设定的搜索到的词的数目
第二行：待搜索的主关键词
第三行：其余包含词1
第四行：其余包含词2
。。      。。

其余包含词可根据情况往下面依次添加，按照如上顺序设定好搜索条件。注意各个词应该顶着文档的左边界输入，左边不要留有空间。

4.开始进行搜索,执行以下命令：
./run_static

5.当搜索到的词的数目达到input.txt中预设定的数量 或者 循环搜索的时候符合要求的词都找完了，程序即会退出。
等待程序退出后，mydata目录下将会多出来两个txt文件，W打头文件的则是搜索到的词，C打头文件的则是对应词的重复频次。
